# pdgapi-diff

A preliminary diff tool for PDG API sqlite database files.

## Installation

Python >= 3.10 is required. For a self-contained installation, you can use a
virtual environment:

``` bash
git clone https://github.com/mjkramer/pdgapi-diff.git
cd pdgapi-diff
python3 -m venv pdgapi-diff.venv
source pdgapi-diff.venv/bin/activate
pip install .
```

After installation, the `pdgapi-diff` program will be available in
`pdgapi-diff.venv/bin` (which gets added to your `PATH` when you activate the
environment).

## Usage

See `pdgapi-diff --help`.
